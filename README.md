# 餐厅服务面试准备网站

这是一个无依赖的静态网站，适合部署到 GitHub Pages，并在手机上复习国际学校餐厅服务/运营助理岗位面试。

## 本地查看

直接打开 `index.html` 即可。页面不需要 npm、构建工具或后端服务。

## 部署到 GitHub Pages

1. 新建一个 GitHub 仓库。
2. 把本文件夹里的所有内容放到仓库根目录，包括 `.github/workflows/deploy.yml`。
3. 推送到 `main` 分支。
4. 在仓库 `Settings -> Pages` 中把 Source 设为 `GitHub Actions`。
5. 等待 Actions 运行完成后，Pages 会给出访问链接。

## 内容模块

- 速记：岗位定位、短板、能力迁移。
- 问答：18 个高频面试题，可搜索、筛选、标记掌握。
- 餐厅英语：8 周学习路线图，英文词句配中文翻译，点击卡片可朗读英文。
- 自我介绍：30 秒、1 分钟、2 分钟版本。
- 转行话术：从幼教转服务运营的核心解释。
- 3 天计划：可打勾保存的复习清单。
- 反问：面试结尾可问的问题。
- 简历：岗位关键词和招聘平台简短版。
